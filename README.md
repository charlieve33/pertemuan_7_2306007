📱 Flutter CRUD Product App (With Image URL)

Aplikasi Flutter sederhana untuk mengelola data produk (Create, Read, Update, Delete) dengan fitur tambahan berupa tampilan gambar produk dari URL internet.

📌 Deskripsi

<img width="754" height="969" alt="image" src="https://github.com/user-attachments/assets/a941f465-e77c-439b-b7bb-cb5c374d1b87" />

Aplikasi ini dibuat menggunakan framework Flutter dengan bahasa pemrograman Dart.
Fungsinya adalah untuk melakukan operasi CRUD pada data produk yang terdiri dari:

Nama produk
Harga produk
Gambar produk (dari URL internet)

Setiap produk yang ditambahkan akan langsung ditampilkan dalam bentuk list lengkap dengan gambar.

✨ Fitur Utama
✅ Menampilkan daftar produk
➕ Menambahkan produk baru
✏️ Mengedit produk
❌ Menghapus produk
🖼️ Menampilkan gambar dari URL
👁️ Preview gambar saat add/edit produk (realtime)
